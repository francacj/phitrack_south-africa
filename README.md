# South Africa Phitrack Shiny App
 
This repository is set up with **renv** for reproducible R environments and includes templates for Word reports and convenience scripts.

## Quick start

### 1. Clone the repository
```bash
git clone https://github.com/francacj/phitrack_south-africa

Unzip the folder.


### 2. Install packages:
Go to folder "phitrack_south-africa" , double click on tool-sa.Rproj.
Load + run: install_packages.R


### 3. Run app:
Load: server.r , ui.r, shiny_run.r. 
Open "shiny_run.r". Run rows one by one.
